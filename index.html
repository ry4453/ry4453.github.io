The unitED Calculator

How to use unitED

Performing Calculations

  1. Enter in numbers: Either through Number buttons or Keyboard into the input field (lowest text field in the display)
    • (Optional): Enter in a predefined unit, in the drop down menu, or continue without units
    • (Optional): Change the entered number to either negative or positive (+/-) button
  2. Select any of the 4 operators: +, -, /, *, either through keyboard inputs or the buttons
  3. Once both the number and operator is selected, select the Equals button to complete the calculation
  4. The calculations will display both in the output field and the input field (with or without units)

Playback and Recording

    In the Menu Bar, select the File option and then select Recording to begin recording
    Recording allows you to record your current inputs, for example inputting 2 + 2 = 4 records all of your button or keyboard inputs. Similarly, Playback displays the recorded instructions.

  1. To start the recording, select the Start option
  2. To stop the recording at any time, select the Stop option

    3. For Playback, once completed the recording

  3. Like recording, select the File option and then select Playback to begin playback
  4. To start playback, select the Start option
  5. To stop playback, select the Stop option

Additional Operations

Delete Button: press the delete button at any time to delete a number from the current input field

Reset and Clear: the reset button sets the current operand and current expression back to empty, while the clear button only clears the current operand

History display: display the current calculation history through the arrow (>) button on the right side of the calculator
